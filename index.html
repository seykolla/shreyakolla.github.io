Learning HTML

Hello this is a website so you can learn how to code HTML.
Did you know that this website was made by using a bit of HTML? HTML stands for HyperText Markup Language.
HTML is used to define the structure and layout of a document on the web by using a variety of attributes and tags.

Now I will start teaching you the basics.
First comes first, this is the <head> tag. Whenever you open a tag you must close it with the forward slash (/).
For example when you open the <head> tag, you must close the </head> tag like that.

<head> </head>

That is how you open and close any tag. The forward slash is important at the closing tag because if you don't close a
tag, the computer will think the tag is still running.
